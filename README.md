<h1 align="center">🏥 Hospital Management System</h1>

<p align="center">
A simple and responsive Hospital Management System built using Flask and modern web technologies to manage patients, appointments, doctors, and hospital records efficiently.
</p>

<hr>

<h2>📌 Overview</h2>

<p>
The Hospital Management System is a web-based application designed to simplify hospital administration and patient management processes. The project focuses on creating an easy-to-use platform for handling appointments, patient records, doctor management, and basic hospital operations.
</p>

<p>
This project was developed using Flask with Jinja2 templating and SQLAlchemy for database management while maintaining a clean and responsive UI using Bootstrap.
</p>

<hr>

<h2>🚀 Features</h2>

<h3>👨‍⚕️ Patient Management</h3>

<ul>
  <li>Patient registration and login</li>
  <li>View patient details and records</li>
  <li>Appointment booking system</li>
  <li>Medical history management</li>
</ul>

<h3>🩺 Doctor Management</h3>

<ul>
  <li>Add and manage doctor profiles</li>
  <li>View appointment schedules</li>
  <li>Manage patient consultations</li>
</ul>

<h3>📅 Appointment System</h3>

<ul>
  <li>Schedule appointments</li>
  <li>Appointment tracking</li>
  <li>Manage appointment status</li>
</ul>

<h3>🏢 Admin Dashboard</h3>

<ul>
  <li>Manage patients and doctors</li>
  <li>Hospital data management</li>
  <li>Monitor appointments and records</li>
</ul>

<h3>🎨 Responsive UI</h3>

<ul>
  <li>Clean user interface using Bootstrap</li>
  <li>Responsive design for multiple devices</li>
  <li>User-friendly navigation and layout</li>
</ul>

<h2>⚙️ Installation</h2>

<h3>1️⃣ Clone the Repository</h3>

<pre>
git clone https://github.com/your-username/hospital-management-system.git
</pre>

<h3>2️⃣ Navigate to the Project Directory</h3>

<pre>
cd hospital-management-system
</pre>

<h3>3️⃣ Install Dependencies</h3>

<pre>
pip install -r requirements.txt
</pre>

<h3>4️⃣ Run the Flask Application</h3>

<pre>
python app.py
</pre>

<h3>5️⃣ Open in Browser</h3>

<pre>
http://127.0.0.1:5000
</pre>
